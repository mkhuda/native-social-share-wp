A lightweight, performance-focused social sharing plugin using the native Web Share API with graceful fallbacks.

== Description ==
Native Social Share adds a clean and simple share button to your posts, powered by the native Web Share API with fallbacks for Twitter, Facebook, and LinkedIn.

- Automatically adds share buttons below or above posts
- Auto dark mode support
- No JavaScript dependencies (zero jQuery)
- Progressive enhancement: native share where supported, graceful fallback where not

== Installation ==
1. Upload the `native-social-share` folder to `/wp-content/plugins/`.
2. Activate the plugin through the “Plugins” menu in WordPress.
3. Visit **Settings → Native Social Share** to configure options.
